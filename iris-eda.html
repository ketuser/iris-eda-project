  • 3D view confirms petal-based separation, with minimal outliers (boxplots).

# An example query was executed to calculate average measurements by species:
query = """
SELECT species, AVG(sepal_length) as avg_sepal_length, AVG(petal_length) as avg_petal_length
FROM df
GROUP BY species
"""
result = sqldf(query)
print(result)
